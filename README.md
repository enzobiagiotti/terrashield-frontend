# TerraShield — Frontend Design

Sistema de monitoramento de desastres naturais via satélite.

## Sobre o projeto

O TerraShield é uma plataforma que utiliza dados reais de satélites NASA e INPE
para monitorar queimadas, enchentes e deslizamentos em tempo real,
emitindo alertas antes que os desastres causem danos.
